Core Architecture & Stack

This section outlines the foundational technologies chosen for the CentsFlow SaaS platform. By separating the frontend framework, authentication provider, database, and styling engine, we achieve a highly scalable and maintainable enterprise-grade application. Interact with the cards below to understand the role of each component in the data flow.

⬛

Next.js

The React framework powering the frontend. Utilizes App Router and Server Actions for secure, server-side data fetching directly from the database.

🔒

Clerk Auth

Enterprise-grade user management. Protects routes via middleware and provides the crucial `userId` needed to isolate financial data.

🗃

Supabase

The PostgreSQL backend. Stores the `expenses` table. Relies on Row Level Security (RLS) to ensure users only query their own records.

🎨

Tailwind CSS

Utility-first styling engine combined with Shadcn-inspired minimalism for a sharp, high-contrast, professional aesthetic.
